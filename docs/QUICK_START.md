## 3. v1.3 旅行攻略

### 普通旅行

```text
/旅行攻略 南京
日期：10月20日—10月22日
人数：2人
预算：双人2500元，不含往返大交通
住宿：未定
节奏：松弛
兴趣：历史、城市空间、生活感
必去：明孝陵、南京博物院、总统府
避开：纯商业打卡街区
```

默认输出：

- 最适合的住宿区域
- 每日时间轴
- 交通和停留时间
- 每个地点为什么值得去
- 地理历史知识点
- 吃饭区域
- 预算估算
- 步行与体力
- 雨天、闭馆、人多替代
- 动态信息核查清单

### 一日游

```text
/一日游攻略 青岛，双人，公共交通，节奏普通，重点看山海和老城
```

### 两天一夜

```text
/两天一夜攻略 济南，预算双人1800元，重点看泉水、老城和本地生活
```

### 城市路线规划

```text
/城市路线规划 北京中轴线，一天，不要特种兵，重点是历史和建筑
```

## 4. v1.3 旅行拍摄攻略

```text
/旅行拍摄攻略 青岛
时间：一天
设备：iPhone 15 Pro Max + DJI Pocket 3
重点：海岸、山地、老城坡度、海风、情侣生活感
节奏：普通
```

额外输出：

- 每日内容主线
- 必拍镜头
- 景别和运镜
- 光线与环境声
- 可拆短视频
- 长 Vlog 章节
- 人物互动位置
- 封面候选画面
- 天气不好时的素材替代

推荐加强版：

```text
/旅行拍摄攻略 南京，两天一夜，设备是 iPhone 15 Pro Max + DJI Pocket 3，重点拍城市地理、历史、情侣隐线和生活感。请给我住宿区域、逐日路线、交通、停留时间、必拍镜头、环境声、短视频选题、长Vlog主线、预算、体力和雨天替代。
```

## 5. v1.3 旅行实时核查

出发前 1—3 天使用：

```text
/旅行实时核查 济南
旅行日期：明天
计划地点：趵突泉、大明湖、山东博物馆、老商埠
```

核查：

- 天气和灾害预警
- 开放时间
- 预约
- 门票
- 交通调整
- 临时闭馆
- 封路施工
- 大型活动和限流

此模式需要联网。无法联网时必须明确标注未完成实时核查。

## 7. 推荐工作流

### 普通旅行

```text
第一步：/城市知识包 城市名
第二步：/旅行攻略 城市名 + 日期 + 时长 + 预算
第三步：/旅行实时核查 城市名 + 日期 + 计划地点
```

### 旅行创作

```text
第一步：/深层地理分析 城市名
第二步：/城市选题生成 城市名
第三步：/旅行拍摄攻略 城市名 + 时间 + 设备
第四步：/城市镜头拆解 具体主题
第五步：/城市脚本生成 具体主题
第六步：/城市资料核查 关键说法
第七步：用 cheat-on-content 打分和复盘
```

## 8. 路线质量检查

每份攻略应检查：

```text
路线可行性：/10
预算可靠度：高/中/低
体力强度：低/中/高
动态信息完整度：高/中/低
拍摄价值：/10
城市理解深度：/10
```

普通节奏下：

- 不频繁横穿城市
- 每天保留 60—90 分钟机动时间
- 每天给出可删地点
- 默认每日步行约 8—12 公里以内
- 超过时必须给删减方案
