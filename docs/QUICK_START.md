# city-geo-history-research 快捷调用手册

## 1. 最常用调用

```text
/城市研究 南京
```

用于快速了解一个城市的地理、历史、文化、空间结构和内容选题。

```text
/城市知识包 南京
```

用于输出完整城市知识包，适合旅行前系统准备。

```text
/地区历史地理 临沂兰山区
```

用于研究区县、城区、街区、乡镇等更小尺度地区。

```text
/城市拍摄资料 济南
```

用于直接服务拍摄，重点输出地点、镜头、动线、选题。

```text
/城市冷知识 青岛
```

用于获取可做短视频的地理历史冷知识。

```text
/城市故事文案 南京
```

用于把资料转成有叙事感的短视频或中视频文案。

---

## 2. 适合你的使用方式

### 旅行前 7 天

```text
/城市知识包 目的地城市
```

目标：先建立整体理解。

### 旅行前 3 天

```text
/城市拍摄资料 目的地城市
```

目标：整理拍摄地点、镜头母题、路线。

### 拍摄前 1 天

```text
/城市冷知识 目的地城市
```

目标：找到短视频切口。

### 剪辑前

```text
/城市故事文案 目的地城市 + 我已经拍到的素材
```

目标：把素材变成故事。

---

## 3. 组合调用案例

### 南京

```text
/城市知识包 南京，重点关注六朝、明城墙、民国、秦淮河、法国梧桐、城市破碎感
```

```text
/城市拍摄资料 南京，素材包括中山陵、明孝陵、南京博物院、夫子庙、秦淮河、德基、总统府
```

```text
/城市故事文案 南京，主题是：见过生死，所以温柔；历经繁华，所以淡泊
```

### 临沂

```text
/地区历史地理 临沂兰山区，重点关注老城、沂河、商贸、物流、本地生活内容
```

```text
/城市冷知识 临沂，找适合本地生活短视频的知识切口
```

### 青岛

```text
/城市知识包 青岛，重点关注海岸、山地、德占时期城市规划、红瓦绿树、啤酒工业
```

## 4. 推荐输出要求补充词

你可以在调用后追加：

```text
请按短视频可拍性排序
```

```text
请优先给我适合 60-90 秒短视频的选题
```

```text
请同时给我横屏长 Vlog 主线
```

```text
请标注资料可信度，高/中/低/待核实
```

```text
请不要写成旅游攻略，要解释城市为什么形成今天的样子
```

```text
请结合我的账号：旅行、地理、历史、情侣隐线、真实生活感
```

---

## 5. 推荐工作流

```text
第一步：/城市知识包 城市名
第二步：/城市拍摄资料 城市名
第三步：/城市冷知识 城市名
第四步：把已拍素材发给 Codex 或 ChatGPT，让它生成脚本
第五步：用 cheat-on-content 给脚本打分、预测和复盘
```
