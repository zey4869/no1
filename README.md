# city-geo-history-research

面向城市旅行、地区研究、地理历史科普、城市文化 Vlog 的 Codex / Claude Code skill。

它的目标不是生成普通旅游攻略，而是把一个城市或地区整理成可用于内容创作的「城市知识包」：

- 地理骨架
- 历史沿革
- 行政变迁
- 城市空间结构
- 文化符号
- 人物与事件
- 拍摄地点
- 短视频选题
- 长 Vlog 主线
- 资料可信度分级

## 安装

```bash
npx -y skills add zey4869/no1 -g --all
```

如果你是本地安装，也可以把 `skills/city-geo-history-research` 复制到：

```bash
~/.codex/skills/city-geo-history-research
```

## 适合的内容类型

- 城市文化短视频
- 城市地理科普
- 历史人文 Vlog
- 旅行前资料准备
- 城市知识库整理
- 地方志/博物馆资料转视频脚本
- 横屏中长视频前期研究

## 核心原则

1. 不做普通攻略，要解释城市为什么形成今天的样子。
2. 不只罗列景点，要建立地理、历史、空间、文化之间的关系。
3. 不把百科当唯一来源，要优先使用政府、博物馆、地方志、学术、文保资料。
4. 每个知识点都要考虑是否能转化成镜头、选题或旁白。
5. 结论要分可信度：高、中、低、待核实。
