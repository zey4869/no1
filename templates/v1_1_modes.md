# city-geo-history-research v1.1 六模式模板合集

这个文件用于快速复制调用格式。

---

## 1. 城市资料采集模板

```text
/城市资料采集 {城市/地区/景区/街区}
重点关注：{地理/历史/行政沿革/空间结构/文化符号/人物事件/文保资料/地方志}
内容用途：{短视频/长Vlog/图文/拍摄前准备}
输出要求：请列出资料优先级、关键词、可转化事实、待核实问题，并标注可信度。
```

### 示例

```text
/城市资料采集 南京
重点关注：六朝、明城墙、民国、秦淮河、法国梧桐、城市破碎感
内容用途：90秒短视频和5-10分钟长Vlog
输出要求：请列出资料优先级、关键词、可转化事实、待核实问题，并标注可信度。
```

## 2. 城市选题生成模板

```text
/城市选题生成 {城市/地区/景区/街区}
重点关注：{主题关键词}
账号方向：旅行、地理、历史、生活感、情侣隐线
输出要求：请给我10条竖屏短视频、3条横屏中视频、1条长Vlog主线，并按可拍性和传播潜力排序。
```

### 示例

```text
/城市选题生成 青岛
重点关注：海岸、山地、德占时期城市规划、红瓦绿树、啤酒工业、老城坡度

输出要求：请给我10条竖屏短视频、3条横屏中视频、1条长Vlog主线，并按可拍性和传播潜力排序。
```

## 3. 城市拍摄执行模板

```text
/城市拍摄执行 {城市/地区/路线}
时间：{半天/一天/两天一夜/三天两夜}
设备：{iPhone/DJI Pocket/相机/麦克风}
重点：{城市地理/历史/情侣隐线/生活感/美食/街区/自然}
输出要求：请给我时间路线、必拍镜头、备用镜头、每个镜头对应的选题、素材冗余策略和拍完后的整理方式。
```

### 示例

```text
/城市拍摄执行 南京
时间：两天一夜
设备：iPhone 15 Pro Max + DJI Pocket 3
重点：城市地理、历史、情侣隐线、生活感
输出要求：请给我时间路线、必拍镜头、备用镜头、每个镜头对应的选题、素材冗余策略和拍完后的整理方式。
```

## 4. 城市镜头拆解模板

```text
/城市镜头拆解 {地点/主题}
视频时长：{60秒/80秒/90秒/3分钟/5分钟}
平台：{抖音/小红书/B站/横屏}
主题：{一句话主题}
风格：真实、细腻、有知识、有情绪、有生活气
输出要求：请输出逐镜头表，包含时间、画面、景别、运镜、旁白/字幕、知识点、备注，并标注必拍/可替代镜头。
```

```text
/城市镜头拆解 明孝陵
视频时长：90秒
平台：抖音
主题：南京的厚重感不是高楼给的，而是几百年沉默的石头给的
风格：真实、细腻、有知识、有情绪、有生活气
输出要求：请输出逐镜头表，包含时间、画面、景别、运镜、旁白/字幕、知识点、备注，并标注必拍/可替代镜头。
```

## 7. 完整流水线模板

```text
第一步：/城市资料采集 {城市名}
第二步：/城市知识包 {城市名}
第三步：/城市选题生成 {城市名}
第四步：/城市拍摄执行 {城市名}，{拍摄时间}，设备是{设备}
第五步：/城市镜头拆解 {具体地点/主题}，{时长}
第六步：/城市脚本生成 {具体主题}，{时长}
第七步：/城市资料核查 {关键历史/地理说法}
第八步：用 cheat-on-content 给脚本打分、预测和复盘
```
