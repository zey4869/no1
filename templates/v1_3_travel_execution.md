# city-geo-history-research v1.3 旅行执行模板

---

## 1. 普通旅行攻略模板

```text
/旅行攻略 {目的地}
出发城市：{可选}
旅行日期：{日期}
游玩时长：{半天/一天/两天一夜/多日}
人数：{人数}
预算：{总预算或人均预算}
往返大交通是否计入预算：{是/否}
住宿：{未定/已定区域}
交通：{公共交通/打车/自驾/混合}
节奏：{松弛/普通/紧凑}
兴趣：{历史/地理/建筑/自然/美食/本地生活}
必去：{地点}
避开：{地点或类型}
体力：{低/普通/高}
输出要求：请给我住宿区域、逐日时间轴、地点间交通、停留时间、每个地点为什么值得去、地理历史知识点、餐饮区域、预算、体力、雨天和闭馆替代、可删地点及实时核查项。
```

## 8. 城市知识转旅行模板

```text
/城市知识包 {城市名}
附加任务：基于这份城市知识包，生成一份{天数}旅行攻略。
要求：路线不是热门景点堆砌，而是按照深层地理、历史主线和城市空间结构安排，让人走完后能理解这座城市为什么成为今天的样子。
```

## 9. 深层地理转路线模板

```text
/深层地理分析 {城市名}
附加任务：根据区位、地形、气候、水文、交通通道和老城扩张逻辑，设计一条{一天/两天}深度旅行路线。每个地点标注对应地理因子、停留时间、交通方式、可拍画面和替代地点。
```

## 10. 路线压缩模板

```text
请把上面的旅行攻略压缩为现实可执行版本：
1. 删除价值最低的30%地点；
2. 每天保留60—90分钟机动时间；
3. 减少跨区折返；
4. 重新计算交通、吃饭和停留时间；
5. 输出最终必去、可选和删除清单。
```

## 11. 预算复核模板

```text
请独立复核上面的旅行预算，不要直接相信我提供的估算。分别计算住宿、市内交通、门票、餐饮、临时消费和备用金，并给出：最低可行预算、普通舒适预算、容易超支的环节、需要实时核查的价格。
```

## 12. 体力复核模板

```text
请复核上面的行程体力：估算每天步行里程、步数、爬坡楼梯、连续户外时间和体力等级。如果普通人难以完成，请删减并给出更合理路线。
```
