# city-geo-history-research v1.2 深层地理因子模板

v1.2 用于把城市地理从“位置、地形、水系”扩展到更深的底层变量：气候、土壤、植被、资源、灾害、交通通道、微气候、生活方式与影像可视化。

---

## 1. 深层地理分析模板

```text
/深层地理分析 {城市/地区/景区/街区}
重点关注：{区位/地形/气候/水文/土壤农业/生态植被/资源产业/灾害风险/交通通道/城市微气候/生活方式}
内容用途：{短视频/长Vlog/图文/拍摄前准备}
输出要求：请解释这些地理因素如何影响城市历史、空间结构、产业、饮食、建筑、生活方式和城市性格。最后给我适合拍摄的画面、短视频选题和资料可信度。
```

### 示例

```text
/深层地理分析 南京
重点关注：长江、秦淮河、低山丘陵、亚热带季风气候、梅雨、夏季湿热、梧桐树荫、明城墙、交通通道
内容用途：90秒短视频和5-10分钟长Vlog
输出要求：请解释这些地理因素如何影响南京的历史、空间结构、饮食、建筑、生活方式和城市性格。最后给我适合拍摄的画面、短视频选题和资料可信度。
```

## 2. 地理因子拆解模板

```text
/地理因子拆解 {城市/地区} {地理因子}
分析方向：{历史影响/空间影响/生活方式影响/产业影响/拍摄转化}
输出要求：请说明这个因子是什么、如何影响历史、如何影响城市空间、如何影响生活方式、如何影响产业、如何被拍出来，并给出至少5条可做短视频选题。
```

### 示例

```text
/地理因子拆解 青岛 海岸和山地
分析方向：历史影响、空间影响、生活方式影响、产业影响、拍摄转化
输出要求：请说明海岸和山地如何影响青岛的老城格局、道路坡度、港口产业、海滨生活、德占时期规划、城市审美，并给出至少5条可做短视频选题。
```

```text

内容用途：短视频和城市性格分析
输出要求：请分析气候如何影响南京的建筑、饮食、街道体感、生活节奏、拍摄季节和城市性格，并给出至少8条短视频选题。
```

## 4. 城市微气候分析模板

```text
/地理因子拆解 {城市/街区} 城市微气候
重点关注：{热岛/河湖降温/山体遮挡/海风/高楼风道/树荫/街巷宽窄/地面材质}
输出要求：请分析人在这里为什么会感觉热、冷、闷、湿、干、风大或阴凉，并说明这些体感如何被拍出来。
```

### 示例

```text
/地理因子拆解 南京 老城微气候
重点关注：城墙、秦淮河、法国梧桐、老街阴影、夏季湿热、夜晚水汽
输出要求：请分析人在南京老城为什么会感到潮湿、阴凉、压低和有历史感，并说明这些体感如何被拍成镜头。
```

## 5. 灾害与城市性格模板

```text
/地理因子拆解 {城市/地区} 灾害与风险
重点关注：{洪水/干旱/台风/寒潮/暴雨/热浪/沙尘/地震/滑坡/风暴潮}
输出要求：请分析这座城市长期在防什么，灾害如何影响城址选择、防御工程、水利设施、建筑形态、城市记忆和地方性格。
```

### 示例

```text
/地理因子拆解 济南 泉水与洪涝
重点关注：泉水、地下水、地势、排水、老城水系、暴雨风险
输出要求：请分析泉水和洪涝如何共同塑造济南的城市空间、生活记忆、旅游表达和短视频选题。
```

## 6. 地理因子转镜头模板

```text
/城市镜头拆解 {地理因子主题}
视频时长：{60秒/80秒/90秒/3分钟}
核心观点：{一句话观点}
输出要求：请把这个地理因子拆成逐镜头表，每个镜头必须标注：画面、景别、运镜、旁白、对应地理因子、可拍地点和替代方案。
```

### 示例

```text
/城市镜头拆解 南京的湿热从哪里来
视频时长：90秒
核心观点：南京的湿热不是单纯天气问题，而是长江水汽、季风、低山丘陵、城市密度和树荫系统共同作用的体感结果。
输出要求：请把这个地理因子拆成逐镜头表，每个镜头必须标注：画面、景别、运镜、旁白、对应地理因子、可拍地点和替代方案。
```
